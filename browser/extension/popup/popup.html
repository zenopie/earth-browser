Enable JavaScript for this destination?

Enabling JavaScript allows this destination to:

  • Run arbitrary code on your machine
  • Fingerprint your browser and hardware
  • Potentially exfiltrate data if your machine has IP connectivity
  • Track you across page loads via local storage
  • Exploit rendering engine vulnerabilities

Reticulum protects your transport. JavaScript bypasses that protection at the endpoint.

⚠

Global JavaScript Enabled

You are enabling JavaScript for ALL destinations. This significantly degrades your privacy and security.

With global JS enabled, any destination you visit can:

  • Fingerprint your browser, hardware, and behavior patterns
  • Execute arbitrary code in your browser
  • Attempt to exfiltrate data through any available network interface
  • Correlate your identity across destinations
  • Exploit browser vulnerabilities for code execution

Earth Browser's transport privacy (Reticulum encryption, initiator anonymity, identity management) remains active, but JavaScript can undermine these protections at the application layer.

This is equivalent to running a standard browser. You lose most of the privacy benefits Earth Browser is designed to provide.
