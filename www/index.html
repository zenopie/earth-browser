How It Works

This page was served by earthserv, a Reticulum HTTP server. Your request traveled through an encrypted Reticulum link — no TLS, no certificate authorities, no IP addresses. Just cryptographic destinations and end-to-end encryption.

What You Get

  • Initiator anonymity — the server cannot see your source address
  • End-to-end encryption — X25519 ECDH + AES-256/ChaCha20
  • Self-sovereign addressing — destinations are locally generated keypairs
  • Medium-agnostic transport — works over LoRa, packet radio, WiFi, TCP/IP, anything
  • No JavaScript required — this page is pure HTML + CSS
