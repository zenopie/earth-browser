About Earth Browser

Architecture

Earth Browser is a fork of Tor Browser that replaces the Tor onion routing network with the Reticulum cryptography-based networking stack.

Three components:

  • earthproxy — SOCKS5-to-Reticulum proxy daemon
  • earthserv — Reticulum HTTP server
  • Browser — hardened Firefox fork with all Tor Browser privacy patches

Reticulum

Reticulum is a public domain (2016) networking protocol by Mark Qvist. It provides end-to-end encryption, initiator anonymity, self-sovereign addressing, and medium-agnostic transport.

It works over LoRa, packet radio, WiFi, Ethernet, TCP/IP tunnels, I2P, serial, and anything with >5 bps and a 500-byte MTU.
